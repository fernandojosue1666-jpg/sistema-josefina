# Plataforma Escolar "Josefina Pimentel"

## Introducción
Desarrollo de una interfaz funcional para la Unidad Educativa "Josefina Pimentel", orientada a simplificar las tareas cotidianas del personal docente y administrativo en el manejo de nóminas y expedientes escolares.

## Lógica de Negocios

### 1. Módulo de Autenticación
Sistema de ingreso intuitivo que asigna niveles de acceso según el rol del usuario, asegurando que cada quien gestione sus tareas de manera eficiente.

### 2. Gestión de Nómina
Herramienta de cálculo automatizada para la nómina del personal. Permite organizar cargos, aplicar deducciones y generar reportes financieros claros.

### 3. Expedientes Estudiantiles
Base de datos interactiva para registrar el desempeño académico, datos personales y progreso de cada estudiante de manera ágil y sencilla.

## Propósito
Optimizar el tiempo de respuesta del personal de la institución mediante procesos digitales automatizados y una interfaz clara.
